第三阶段：随机抽样算法（核心）


请帮我实现一个“视频随机抽样算法”。

目标：

根据目标时长，从素材池中随机抽取多个视频片段。

规则：

1. 单个片段最长只能使用 2 秒

2. 如果原视频时长超过 2 秒：

随机截取其中一段 2 秒内容。

例如：

原视频：
3.7 秒

则：

随机 start：
0 ~ 1.7

duration 固定：
2 秒

3. 如果原视频时长小于 2 秒：

直接使用整个视频。

4. 持续随机抽取：

直到：

total_duration >= target_duration

5. 禁止连续两次抽到同一个视频

6. 返回 clips plan

返回结构：

[
  {
    "file": "xxx.mp4",
    "trim_start": 0.3,
    "trim_duration": 2.0,
    "original_duration": 3.7
  }
]

1. 不要在这个阶段做 ffmpeg 导出

2. 仅生成剪辑计划

3. 代码必须模块化

4.  必须支持后续批量生成

请生成完整 Python 实现。
